Obs.: Toda vez que um dispositivo é inicializado é escolhida uma Fog aleatória do conjunto de Fogs (Se for 10, então uma das 10), isso é pré-configurado.

Handshake:
	topic: fog/{idFog} | Subscriber -> Fog // Publisher -> Device
	data: "id" //id do dispositivo
	topic: device/{idDevice} | Subscriber -> Device // Publisher -> Fog
	data: "topic_thread: fog/{idFog}/{threadid}" //Nos dados temos o tópico que o dispositivo deve publicar para sempre.

Envio de dados pelo dispositivo:
	topic: fog/{idFog}/{threadid} Subscriber -> FogThread // Publisher -> Device
	data: 
		"{
		    id: *, //id do dispositivo
		    temperature: *, 
		    fcardiac: *, 
		    frespiratory: *, 
			bloodoxygen: *, 
			arterialpressure: *,state: *
		}"
